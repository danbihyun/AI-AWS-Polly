# Chapter 09. 운영, 보안, 비용 최적화

## 학습 목표
- 서버리스 운영 시 모니터링/보안/비용 관리를 수행한다.

## 운영 체크리스트
- CloudWatch Logs로 실패율/지연 관찰.
- Lambda timeout/memory 튜닝.
- Polly 호출량 및 문자 수 모니터링.

## 보안 체크리스트
- 최소 권한 IAM 정책.
- Function URL 인증 방식(IAM 또는 NONE+WAF) 검토.
- S3 버킷 퍼블릭 차단 유지.

## 비용 체크리스트
- 캐싱 가능한 음성은 재사용.
- 짧은 텍스트는 묶어서 호출.
- 불필요한 오브젝트 자동 삭제.
