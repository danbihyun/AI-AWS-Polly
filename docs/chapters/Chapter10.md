# Chapter 10. 캡스톤: 텍스트→MP3 저장/재생 완성

## 학습 목표
- 지금까지의 구성을 하나의 사용자 시나리오로 통합한다.

## 최종 시나리오
1. FE에서 텍스트 입력
2. Lambda API 호출
3. Polly로 MP3 생성
4. S3 업로드
5. presigned URL 응답
6. 오디오 플레이어에서 즉시 재생

## 검증 항목
- 한국어 보이스 정상 합성 여부
- S3 오브젝트 생성 여부
- URL 만료 전 재생 여부
- 오류 상황(빈 텍스트, 권한 오류) 처리 여부
